学生工具助手系统

一款轻量全能的学生专属工具箱，助力简化日常学习流程、高效管理作业任务，全方位提升学业效率。

✨ 核心功能

• 作业追踪器：记录作业截止日期、设置提醒，按课程分类任务，杜绝遗漏提交。

• 学习计时器：支持番茄工作法与自定义时长模式，帮助保持专注学习状态。

• 笔记管理器：按学科分类笔记、添加标签实现快速检索，支持 TXT/Markdown 等多格式导出。

• 成绩计算器：输入课程学分与分数，自动核算 GPA 与加权平均分。

• 资源库：集中收纳、管理学术资源、教材电子版及网课链接，打造专属学习资源中心。

🛠️ 技术栈

• 前端：HTML/CSS/JavaScript（如需搭建交互界面，可选用 Vue.js 框架）

• 后端：Python（Flask/Django 框架） / Node.js（Express 框架）

• 数据库：SQLite（轻量部署场景） / MySQL（多用户协作场景）

• 部署方式：支持 Docker 容器化部署，可本地运行，也能部署至 GitHub Pages、Heroku 等云平台

🚀 快速开始

1. 克隆代码仓库
git clone https://github.com/你的用户名/student-tool-assistant-system.git
cd student-tool-assistant-system
2. 安装依赖包
# Python 后端环境
pip install -r requirements.txt
# Node.js 后端环境
npm install
3. 启动应用程序
# Python (Flask 框架)
python app.py
# Node.js (Express 框架)
npm start
4. 在浏览器访问 http://localhost:5000（或后端配置的指定端口）即可使用系统
